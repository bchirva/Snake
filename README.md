### Simpe, classic Snake game.
Written in C++ with [SFML](https://sfml-dev.org).

##### Requirements:
* C++ compiler that support 14 standard;
* CMake at least 3.5.2 version;
* SFML (v2.5 is used in this project)

##### Default Control:
Command | Key 
------- | ----------- 
Turn Up | <kbd>W</kbd> 
Turn Down | <kbd>S</kbd> 
Turn Left | <kbd>A</kbd>
Turn Right | <kbd>D</kbd> 
Pause | <kbd>Space</kbd> 
Exit | <kbd>Esc</kbd>
